# WSQ — What It Means

## What Is WSQ?
WSQ stands for **Workforce Skills Qualifications** — Singapore's national credentialling system for workforce training. Courses that are WSQ-certified are:
- Accredited by SSG (SkillsConnect / Skills & Competency)
- Eligible for government subsidies (SSG, SkillsFuture Credit, UTAP)
- Nationally recognised across employers in Singapore

## What Do You Get?
A **Statement of Attainment (SOA)** — a nationally recognised e-certificate issued via myskillsfuture.sg. See [[policies/certification]].

## WSQ vs Non-WSQ vs Private
| Type | Subsidy Available | Certificate |
|---|---|---|
| **WSQ** | Yes — SSG, SFC, UTAP eligible | SOA via myskillsfuture.sg |
| **Non-WSQ Certifiable** | No SSG subsidy | Certificate from provider |
| **Private** | No subsidy | Certificate from provider |

Coursemology offers all three types. Check each course page — the `course_type` tag indicates which applies.

## TGS Code
Every WSQ course has a **TGS code** (e.g. TGS-2022016432). You need this code to:
- Search for the course on myskillsfuture.gov.sg
- Submit a SkillsFuture Credit claim
